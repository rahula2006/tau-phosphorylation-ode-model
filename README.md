# tau-phosphorylation-ode-model
A systems biology ODE model of Tau phosphorylation dynamics in Alzheimer’s disease. Simulates the kinase–phosphatase tug‑of‑war (GSK3B, CDK5, CSNK2A1, AKT1 vs PP2A with PIN1 facilitation) and shows how AD‑like perturbations (PP2A/PIN1 loss, GSK3B hyperactivity) shift Tau toward a hyperphosphorylated steady state.
